# USE CASE: 3 Produce a Report on the Salary of Employees in My Department

## CHARACTERISTIC INFORMATION

### Goal in Context

As a *department manager* I want *to produce a report on the salary of employees in my department* so that *I can support financial reporting for my department.*

### Scope

Company.

### Level

Primary task.

### Preconditions

The department is known. Database contains current employee salary data.

### Success End Condition

A report is available for the manager to use for departmental financial reporting.

### Failed End Condition

No report is produced.

### Primary Actor

Department Manager.

### Trigger

A request or requirement for departmental financial reporting arises.

## MAIN SUCCESS SCENARIO

1. Department manager initiates a request to access salary information for their department.
2. Department manager identifies the department for which data is needed.
3. Department manager retrieves current salary data for all employees in the department.
4. Department manager compiles and reviews the report.
5. Report is used for internal departmental financial reporting.

## EXTENSIONS

3. **Department data is incomplete or unavailable**:
    1. Department manager contacts HR or IT to resolve the data issue.

## SUB-VARIATIONS

None.
